# 01-Projects - 项目目录

> **这是什么**：有**明确截止日期**的任务和事项
> **什么时候用**：启动新任务、跟踪项目进展、项目完成后归档
> **存放内容**：项目文档、计划、进展记录、交付物
> **完成标准**：达成目标后标记完成，移动到 Archive/

**与 Areas 的区别**：Projects 有 deadline，Areas 是长期维护

## ✅ 完成标准

项目完成后：
1. 在此标记状态为 ✅
2. 将项目档案移动到 `memory/projects/`
3. 更新本索引指向归档位置
4. 记录经验教训到 `memory/lessons/`
