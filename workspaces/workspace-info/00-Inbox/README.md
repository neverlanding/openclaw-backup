# 00-Inbox - 收件箱

> **这是什么**：临时存放**待处理**的文件和任务
> **什么时候用**：新文件进来、快速记录、待分类内容
> **处理规则**：24小时内必须清空，不能长期堆积
> **处理完后**：移动到 Projects/Areas/Resources/Archive 对应位置

## 处理流程

1. **接收** → 新文件放入此目录
2. **分类** → 判断属于 P/A/R/A 哪个分类：
   - **Projects** → 有明确截止日期的任务（如配置优化专项）
   - **Areas** → 长期维护的领域（如系统运维、多助手协调）
   - **Resources** → 可复用的参考资料（如脚本、报告）
   - **Archive** → 已完成的待归档
3. **移动/链接** → 将文件移到正确位置
4. **归档记录** → 在本文件记录处理历史
