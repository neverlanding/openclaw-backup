## 安装方法

```bash
# 方法1: 使用 clawhub CLI
npx clawhub@latest install <skill-name>

# 方法2: 从 ClawHub 网页下载
# 访问 https://clawhub.ai/<author>/<skill-name>
# 下载后解压到 ~/.openclaw/skills/
```
